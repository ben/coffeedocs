Class: elemica.Paginator

Defined in: portal/src/main/webapp/coffee-script/paginator.coffee

Overview

Utility class to help with pagination.

Exposed properties

  • pageInfo (array) the full set of pages that are available for viewing.

  • trimmedPageInfo (array) If pageInfo is too large, this contains a "trimmed" set. The first, last, and active pages will always be shown, but the ranges in between may be replaced with ellipses.

Contained objects

The objects contained in the pageInfo and trimmedPageInfo look like this:

  • number (string) the text to display ("First", "1", "â€¦", etc.)
  • changePage (function) call when this page should be activated
  • isActive (observable/bool) true when this page is the current one
  • isDisabled (boolean) true when this page cannot be changed to

Constructor Details

# (void) constructor(sizeObservable, currentPageObservable, entriesPerPage)

Parameters:

  • sizeObservable ( ko.observable ) — Contains the full count of items
  • currentPageObservable ( ko.observable ) — Contains the 0-indexed current page. Paginator may write to this observable.
  • entriesPerPage ( Number ) — The number of items on each page.

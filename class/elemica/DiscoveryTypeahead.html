# (void) tokenizeJSON(value)

We store the result String as a JSON array without the []s. To split it into its correct tokens, we parse the string fully, then return the stringified version of each object individually. If we attempted a non- parser-based split, like splitting on commas, we'd end up with values like '{type:"Company"' and 'value:"Elemica"}'. This breaks the ability to tell if a value is already in the contents, as well as breaking the ability to remove an item.

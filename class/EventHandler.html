Class: EventHandler

Defined in: portal/src/main/webapp/coffee-script/event-handler.coffee

Overview

EventHandler provides an easy convenience function, events, which takes an object whose keys are events and whose values are typically functions to bind for those events on the document element. The values can also be a [selector, handler] pair, where the selector is a string indicating the selector that must match the event target in order for the handler to run. This latter form is the same as calling $(document).on(event, selector, handler)

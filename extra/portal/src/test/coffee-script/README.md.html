Knockout model tests

You've found the tests! To run these from SBT:

> project portal
> test

…but while you're writing them, it's probably easier to have them run all the time:

$ cd portal/src/test/coffee-script
$ npm install
$ cake watch

Or if you prefer, under SBT:

> ~ testOnly *Coffee*

How to write a testable class

Your class should be available in the elemica namespace. This is defined for you by elemica-base.coffee (browser) and test-environment.coffee (tests), so just drop them in there:

class elemica.FooBar
  doSomething = -> # …

How to test something

Then you're ready to write your spec file. Let's call it foo-bar-tests.coffee:

require 'test-environment'
require 'foo-bar'

describe 'FooBar', ->
  describe '#baz', ->
    it 'should do something', ->
      fb = new elemica.FooBar
      fb.doSomething()
      assert.isTrue fb.didSomething()

The tests run under Mocha, and that assert.isTrue comes from Chai.

Method Details

~ (void) elemica.markAsLoading($container, onComplete)

This function will mark the given container as loading. This means we run a heuristic to determine the best place to put a loading spinner, and disable any inputs within the contaainer. We also mark the container with a class, 'submitting'.

The second thing this does is install an event handler on the container element, watching for the 'form-submission-completed' event. When seen, the form is restored to its original state. A callback can be passed that will be invoked when the form submission is completed for any additional resetting tasks you may need to do.
